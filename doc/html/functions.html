    write_iterator_range(std::basic_ostream<CharT, Traits>& out, InputIterator first, Sentinel last, Delimiter const& delim);

Description

The non-delimiter version writes each element of the sequence defined by [first, last) to the output stream out, using the insertion operatior (operator<<). The delimiter version does the same, but between each element it writes delim (using out << delim). Both versions return a write_iterator_range_result_t<InputIterator> with details about the write.

Writing stops immediately when the output stream is false (after conversion to bool). Otherwise, it continues until the iterator to the next element compares equal to the sentinel object.

Header

#include <boost/rangeio/write_iterator_range.hpp>

Type requirements

Parameter Type requirements
InputIterator Must satisfy InputIterator concept.
Sentinel Must be EqualityComparable with InputIterator.
Delimiter Objects of type Delimiter must be streamable to objects of type std::basic_ostream<CharT, Traits>.
CharT Must be usable in std::basic_ostream<CharT, Traits>.
Traits Must be usable in std::basic_ostream<CharT, Traits>.

Arguments

Name Type Description
out std::basic_ostream<CharT, Traits>& Reference to an output stream that the elements of the sequence defined by [first, last) will be written to.
first InputIterator Iterator to the start of the sequence to be written.
last Sentinel Sentinel object used to detect (via equality comparison with an iterator) when the end of the sequence is reached.
delim Delimiter Optional delimiter object to be written between each element of the input sequence.

Preconditions

[first, last) must describe a valid sequence.

Example

#include <boost/rangeio/write_iterator_range.hpp>

void f()
{
  std::vector<int> v{1, 2, 3, 4, 5};
  
  // Of course, it would be better to use auto
  boost::rangeio::write_iterator_range_result_t<std::vector<int>::iterator> res1 =
    boost::rangeio::write_iterator_range(std::cout, v.begin(), v.end()); // writes "12345" to cout
  
  assert(res1.count == 5); // 5 elements were written
  assert(res1.next == v.end()); // all elements were written
  
  using boost::rangeio::write_iterator_range;
  auto res2 = write_iterator_range(std::cout, v.begin() + 1, v.begin() + 4, ", "); // writes "2, 3, 4" to cout
  
  assert(res2.count == 3); // 3 elements were written
  assert(res2.next == (v.begin() + 4)); // all elements were written
}

C++98 compatibility

The only limitation in C++98 is that if the delimiter is an rvalue, it will be const. Non-const rvalue delimiters are not supported in C++98.
